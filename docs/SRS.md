# SRS / Техническое задание — ООО «СистемаКонтроля»

## 1. Цель
Монолитное веб‑приложение (Django) для централизованного управления дефектами на строительных объектах: от регистрации дефекта и назначения исполнителя до контроля статусов и формирования отчётности.

## 2. Роли и права
- **Инженер**: создаёт дефекты, прикрепляет файлы, меняет статус только своих задач (по workflow).
- **Менеджер**: полный CRUD по проектам/дефектам, назначает исполнителей, закрывает/отменяет.
- **Руководитель/Заказчик (наблюдатель)**: чтение + аналитика и отчётность.
